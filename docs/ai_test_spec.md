# テスト仕様書（自動販売機システム）

## 1. 概要
本ドキュメントは自動販売機システムの各機能に対するテスト仕様を示す。

| F01    | 商品一覧表示   | 商品一覧が正しく表示される |
| F02    | 商品登録       | 新規商品が登録できる       |
| F03    | 商品情報編集   | 商品情報が編集できる       |
| F04    | 商品削除       | 商品が削除できる           |
| F05    | 在庫数自動更新 | 購入時に在庫が減る         |
| F06    | 商品入れ替え   | 商品の入れ替えができる     |
| F07    | 商品選択       | 商品が選択できる           |
| F08    | 数量入力       | 数量が正しく入力できる     |
| F09    | 合計金額表示   | 合計金額が正しく表示される |
| F10    | 現金投入受付   | 現金が正しく受付される     |
| F11    | 釣銭計算・返却 | 釣銭が正しく計算・返却される |

## 3. テストケース詳細

### F01: 商品一覧表示
- テスト項目: 商品一覧が正しく表示される
- 手順: 商品登録後、一覧画面を表示
- 期待結果: 登録した商品が全て表示される

### F02: 商品登録
- テスト項目: 新規商品が登録できる
- 手順: 商品情報を入力し登録
- 期待結果: 入力内容の商品が一覧に追加される

### F03: 商品情報編集
- テスト項目: 商品情報が編集できる
- 手順: 既存商品の情報を編集
- 期待結果: 編集内容が反映される

### F04: 商品削除
- テスト項目: 商品が削除できる
- 手順: 商品を選択し削除
- 期待結果: 商品が一覧から消える

### F05: 在庫数自動更新
- テスト項目: 購入時に在庫が減る
- 手順: 商品を購入
- 期待結果: 在庫数が1減る

### F06: 商品入れ替え
- テスト項目: 商品の入れ替えができる
- 手順: 既存商品を新商品に入れ替え
- 期待結果: 新商品が一覧に表示される

### F07: 商品選択
- テスト項目: 商品が選択できる
- 手順: 商品を選択
- 期待結果: 選択した商品が購入対象となる

### F08: 数量入力
- テスト項目: 数量が正しく入力できる
- 手順: 数量を入力
- 期待結果: 入力した数量が反映される

### F09: 合計金額表示
- テスト項目: 合計金額が正しく表示される
- 手順: 商品・数量を選択
- 期待結果: 合計金額が正しく計算・表示される

### F10: 現金投入受付
- テスト項目: 現金が正しく受付される
- 手順: 現金を投入
- 期待結果: 投入金額が反映される

### F11: 釣銭計算・返却
- テスト項目: 釣銭が正しく計算・返却される
- 手順: 購入金額以上の現金を投入
- 期待結果: 釣銭が正しく返却される

### F12: 購入確定
- テスト項目: 購入が確定し在庫・ログ更新
- 手順: 購入確定操作を行う
- 期待結果: 在庫・売上・ログが更新される

### F13: 複数客対応
- テスト項目: 複数客が順次利用できる
- 手順: 連続して複数人が購入
- 期待結果: 各客の操作が独立して処理される

### F14: ログ保存
- テスト項目: ログがCSVに保存される
- 手順: 操作・エラー発生後、ログファイルを確認
- 期待結果: 操作・エラー内容がCSVに記録されている
